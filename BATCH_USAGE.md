# 매뉴얼 크롤러 일괄 처리(Batch) 사용법

새로 추가된 **"한방에 다 하는 기능"** 사용법입니다.

### 1단계: 프롬프트 생성 및 복사
1. 좌측 사이드바의 **[📥 로드 & 프롬프트 복사]** 버튼을 클릭합니다.
2. 현재 목록의 모든 기사 내용이 로드되고, LLM용 프롬프트가 자동으로 클립보드에 복사됩니다.
   *(알림창이 뜨면 확인을 누르세요)*

### 2단계: LLM 분석 요청
1. ChatGPT, Claude, Gemini 등의 LLM 채팅창에 붙여넣기(`Ctrl+V`) 합니다.
2. AI가 JSON 형식으로 분석 결과를 출력할 때까지 기다립니다.
3. 출력된 **JSON 전체를 복사**합니다.

### 3단계: 일괄 적용 및 자동 저장
1. 크롤러 화면으로 돌아와 **[⚡ 일괄 적용 (Apply Batch)]** 버튼을 클릭합니다.
2. 텍스트 창에 복사한 JSON을 붙여넣습니다.
3. **[파싱 적용]** 버튼을 누릅니다.
4. **"자동으로 검산 및 저장(Verify & Save) 하시겠습니까?"** 확인창이 뜨면 **[확인]**을 누릅니다.
   * **[확인]**: 모든 항목에 대해 1) 점수 반영, 2) 검산(Verify), 3) 파일 저장(Inject)이 순차적으로 자동 실행됩니다.
   * **[취소]**: UI에만 점수가 반영되고 파일로 저장되지는 않습니다. (눈으로 확인 후 개별 저장 시 선택)

---

### (옵션) 배치 검사기 (Inspector Mode) 사용법
하나씩 꼼꼼하게 보면서 처리하고 싶다면 **[🔍 배치 검사기]**를 사용하세요.

1. **[🔍 배치 검사기 (Inspector)]** 버튼 클릭 (새 창이 뜹니다).
2. 왼쪽 리스트에서 기사를 선택합니다.
3. **Step 1 [📋 복사]** 버튼으로 해당 기사 전용 프롬프트를 복사하여 LLM에 질문합니다.
4. LLM 답변을 **Step 2** 창에 붙여넣습니다.
5. **[💾 검증 및 저장]** 버튼을 누르면 저장되고, 자동으로 다음 기사로 넘어갑니다.
